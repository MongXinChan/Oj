滑动窗口本质上就是一个双指针。

209.[长度最小的子数组](https://leetcode.cn/problems/minimum-size-subarray-sum/solution/biao-ti-xia-biao-zong-suan-cuo-qing-kan-k81nh/) 

我们可以枚举子数组的左端点向右扩展，也可以枚举数组的右端点向左扩展

其时间复杂度为$O(N^2)$，这是因为我们没有利用数组元素都是正的原因。我们可以保留上一次计算的记过，比如说我们之前枚举右端点的时候，向左扩展，扩展到了其大于等于7，在枚举下一个数的时候，不用老实地向左扩展，而是在其基础上再加4。

并且可以让其左端点向右缩小。

其时间复杂度为$O(N)$

    def minSUbArrayLen(self,target:int,nums:List[int])->int:
        n=len(nums)
        ans=n+1
        s=0
        left=0
        #left<=right 不写的原因是因为条件说target是一个非负自然数
        for right,x in enumerate(nums): #x:nums[right]
            s+=x
            while s- nums[left]>=target:
                s-=nums[left]
                left++
           	if s>=target:
                ans=min(ans,right-left+1)
        return ans if ans<= n else 0 # ans <= n?ans:0;
